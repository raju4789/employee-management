# employee-management

A simple app to maintain employee registry.

## BUILT USING:
React Hooks, Spring Boot, Role based authorisation using JWT and Spring Security, Spring Data JPA, H2(Inmemory SQL database), REST, HTML, CSS, Bootstrap.
